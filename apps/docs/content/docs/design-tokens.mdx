OpenOS marketing and apps share a **Cohere-inspired** system:

- **Canvas:** white (`#ffffff`).
- **Ink:** primary text (`#212121`).
- **Primary / structural:** deep neutral (`#17171c`).
- **Deep green:** brand bands (`#003c33`).
- **Soft stone:** panels (`#eeece7`).
- **Action blue:** links and primary actions (`#1863dc`).
- **Coral:** accent (`#ff7759`).
- **Muted:** secondary text (`#93939f`).

Typography uses **Space Grotesk** (display) and **Inter** (body) with system fallbacks. Radius: large cards `22px`, pill CTAs `32px`. Section spacing `80px` on wide layouts.

See the repository root `DESIGN.md` for the full specification.
