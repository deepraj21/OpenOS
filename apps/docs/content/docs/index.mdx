OpenOS is a **developer SDK and runtime** for building agentic operating systems: scheduling, memory, tools, LLM adapters, and packaging. Phase 1 ships the TypeScript kernel and SDK; Phase 2 adds the registry, docs, composer, MCP, and A2A.

Use the sidebar to explore quick start, architecture, and ecosystem topics.
